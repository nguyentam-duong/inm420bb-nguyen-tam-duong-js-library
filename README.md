# Vietnam Travel Site

A modern, responsive travel/vacation website for Vietnam, built as a student project.

## 🌏 Features

- **Hero carousel** with beautiful images (Glide.js)
- **Tour packages grid** with discount badges and responsive cards
- **Why Vietnam** section with icons and benefits
- **Animated scroll effects** (AOS)
- **Interactive chart** for travel stats (Chart.js)
- **Interactive map** showing Vietnam (Leaflet.js)
- **FAQ accordion** for common questions
- **Sticky header** with burger menu for mobile
- **Mobile-first responsive design**
- **Consistent, premium code style**

## 🚀 Technologies & Libraries Used

- [Glide.js](https://glidejs.com/) – Hero image carousel
- [AOS](https://michalsnik.github.io/aos/) – Animate On Scroll
- [Chart.js](https://www.chartjs.org/) – Travel stats chart
- [Leaflet.js](https://leafletjs.com/) – Interactive map
